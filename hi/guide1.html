Hướng dẫn Dualboot iPod touch 4 iOS 6 -> iOS 7

Giới thiệu

Ngày phát hành: 2011 sử dụng IPSW 3,1 (7.1.2), 2012 sử dụng IPSW 3,3 (7.1.2). Để thực hiện dualboot, bạn cần chuẩn bị rootfs trước tiên.

Chuẩn bị

  • mkdir /mnt1 & mkdir /mnt2

Thiết bị của mình có dung lượng 32GB. Để tạo phân vùng cho hệ điều hành thứ 2, mình sẽ giữ lại 16GB, phân vùng thứ 2 cũng là 16GB. Tính toán: 16 x 1024 x 1024 x 1024 (đơn vị byte). Lưu ý để trống khoảng 2GB để đảm bảo hoạt động ổn định.

Tạo phân vùng

  • Nếu chưa phân vùng, sử dụng: hfs_resize /private/var (mình đã phân vùng nên bỏ qua bước này).
  • gptfdisk /dev/rdisk0s1
  • - p enter i enter 2 enter: Lấy Partition unique GUID, sao chép lại.
  • - d enter 2 enter n enter enter: Dung lượng phân vùng (mình dùng: 17179869184/8192 + 169368 = 2266520).
  • - enter enter
  • - c enter 2 enter Data enter
  • - x enter a enter 2 enter 48 enter enter
  • - c enter 2 enter: Nhập GUID vừa sao chép.
  • - s enter 4 enter m enter n enter 3 enter enter: Tính toán (byte của UDZO.dmg / 8192 + first sector 2266524 = 2462497).
  • - enter enter
  • - c enter 3 enter iOS7SYSTEM
  • - n enter 4 enter enter: Last sector default - 5 (mình là 3903487).
  • - enter enter
  • - c enter 4 enter iOS7DATA
  • - p enter w enter Y enter: Hoàn tất phân vùng.
  • sync
  • reboot (nên reboot để chắc chắn)

Kết nối SSH

  • /sbin/newfs_hfs -s -v System -J -b 8192 -n a=8192,c=8192,e=8192 /dev/disk0s1s3
  • /sbin/newfs_hfs -s -v Data -J -b 8192 -n a=8192,c=8192,e=8192 /dev/disk0s1s4
  • mount_hfs /dev/disk0s1s4 /mnt2
  • Copy UDZO.dmg vào /mnt2 bằng afc2 hoặc scp.
  • asr restore --source /mnt2/UDZO.dmg --target /dev/disk0s1s3 --erase
  • umount /mnt2
  • mount_hfs /dev/disk0s1s3 /mnt1
  • /sbin/newfs_hfs -s -v Data -J -P -b 8192 -n a=8192,c=8192,e=8192 /dev/disk0s1s4
  • mount_hfs /dev/disk0s1s4 /mnt2
  • mv -v /mnt1/private/var/* /mnt2
  • rm -rf /mnt2/mobile/Library/PreinstalledAssets/*
  • rm -rf /mnt2/mobile/Library/Preferences/.GlobalPreferences.plist
  • mkdir /mnt2/keybags
  • cp -a /var/keybags/systembag.kb /mnt2/keybags
  • umount /mnt2
  • nano /mnt1/private/etc/fstab
